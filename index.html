Сохранённые маршруты

Настройки · только администратор

Важно про безопасность. Это статичное приложение в браузере. Разделение ролей и PIN-коды защищают от случайных правок и любопытных, но не являются настоящей защитой: код открыт, и технически подкованный пользователь может их обойти. Ключи и токены, введённые здесь, хранятся только в этом браузере (localStorage) и не попадают в data.json. Для настоящего контроля доступа нужен закрытый хостинг с авторизацией или прокси-сервер.

Ключ openrouteservice

Используется для маршрутов по дорогам и буфера по времени. Без ключа маршрут рисуется прямыми, но ссылка для водителя в Google Maps работает всегда.

Автосохранение в GitHub

Коммитит data.json прямо в репозиторий через GitHub API. Нужен Fine-grained Personal Access Token с правом Contents: Read and write на этот репозиторий. Токен остаётся только в этом браузере.

Уровни доступа (PIN)

Пустой PIN = роль доступна без кода. Задай PIN для admin и logist, чтобы повышение роли требовало кода. Водитель — без кода. PIN хранится только в этом браузере.
