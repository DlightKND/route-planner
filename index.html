Экономика выезда

Настройки · только администратор

Про безопасность. Это статичное приложение в браузере; на публичном GitHub Pages исходник и data.json открыты всем. Логины и 6-значные PIN разграничивают роли и спасают от случайных правок, но не являются защитой: хэши PIN видны в репозитории, а 6 цифр перебираются мгновенно. Ключ ORS и токен GitHub хранятся только в этом браузере. Для настоящего контроля нужен закрытый бэкенд.

Параметры выезда и тарифы

Используются во вкладке «Экономика». Хранятся в data.json и синхронизируются автокоммитом — едины для всех.
Себестоимость (для оценки рентабельности во вкладке «Экономика»):

Тема по умолчанию для новых пользователей

Цвета, которые получит каждый новый пользователь при создании. Существующие могут менять свой вид сами через «🎨 Вид». Хранится в data.json.

Пользователи

Логин + 6-значный PIN + роль. Первый вход по умолчанию: admin / 000000 — сразу смени PIN.

Ключ openrouteservice

Для маршрутов по дорогам, альтернатив и изохрон. Без ключа маршрут — прямыми, и экономику по времени не посчитать.

Автокоммит в GitHub

Коммитит data.json в репозиторий через GitHub API. Fine-grained PAT с правом Contents: Read and write. Токен — только в этом браузере.

Вход

Внешний вид (для текущего пользователя)
